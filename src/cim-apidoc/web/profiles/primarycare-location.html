Resource PrimaryCare-Location

The official URL for this profile is: 

http://endeavourhealth.org/fhir/StructureDefinition/primarycare-location

Maturity level: Initial development.

Description

A physical location where patients are seen, where services exist, from which requests for activities are sent or any other place of interest to an organisation.

Definition

This profile is derived from Location.
