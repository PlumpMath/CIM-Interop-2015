Value set PrimaryCare-MedicationSupplyType

The official URL for this value set is: 

http://endeavourhealth.org/fhir/ValueSet/primarycare-medication-supply-type

Maturity level: Draft (for review).

Description

Indicates the legal/contractual nature of a prescription. This valueset references the vocabulary for EhrSupplyType in the NPfIT Message Implementation Manual 7.2.02.

Definition

This valueset is comprised of the following codes from external code system http://snomed.info/sct:

CodeDisplay

394824001Private Prescription
394825000ACBS Prescription
394826004OTC sale
