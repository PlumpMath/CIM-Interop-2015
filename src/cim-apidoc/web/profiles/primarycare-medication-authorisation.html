Description

Primary Care medication authorisation.

Medication authorisations, or courses, are “templates” for a prescription. Active or past. They contain enough information for a prescription to be issued or for it to be transcribed into another record as a reference (e.g. secondary care noting the primary care information would be a subset). What it is, who authorised it, when it was authorised, reference to last issue, overall compliance for that course, special instructions etc.
