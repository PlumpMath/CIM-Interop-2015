Value set PrimaryCare-EthnicCategory

The official URL for this value set is: 

http://endeavourhealth.org/fhir/ValueSet/primarycare-ethnic-category

Maturity level: Draft (for review).

Description

The ethnicity of a patient, as specified by the patient. This valueset is comprised of codes from references NHS Data Dictionary: Ethnic Category Code.

Note: This valueset defines an inline code system for an NHS code set. This is a temporary provision while an NHS FHIR defined code system becomes available.

Reference: http://www.datadictionary.nhs.uk/data_dictionary/attributes/e/end/ethnic_category_code_de.asp?shownav=0.
