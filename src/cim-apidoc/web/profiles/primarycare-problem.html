Resource PrimaryCare-Problem

The official URL for this profile is: 

http://endeavourhealth.org/fhir/StructureDefinition/primarycare-problem

Maturity level: Draft (for review).

Description

Primary Care problem.

Use to record detailed information about the onset of a health problem episode recognised by a clinician.

This profile is derived from Condition.
