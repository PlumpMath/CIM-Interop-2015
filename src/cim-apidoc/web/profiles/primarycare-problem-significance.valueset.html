Value set PrimaryCare-ProblemSignificance

The official URL for this value set is: 

http://endeavourhealth.org/fhir/ValueSet/primarycare-problem-significance

Description

List of codes that can be used to indicate the clinical significance of a Problem

Definition

This valueset is comprised of the following codes from external code system http://snomed.info/sct:

CodeDisplay
