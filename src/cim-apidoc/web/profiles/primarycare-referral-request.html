Resource PrimaryCare-ReferralRequest

The official URL for this profile is: 

http://endeavourhealth.org/fhir/StructureDefinition/primarycare-referral-request

Maturity level: Draft (for review).

Description

Primary Care referral request.

Used to record and send details about a request for referral service or transfer of a patient to the care of another provider or provider organisation.

Definition

This profile is derived from ReferralRequest.
