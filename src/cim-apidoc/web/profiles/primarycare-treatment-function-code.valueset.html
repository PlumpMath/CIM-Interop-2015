The official URL for this value set is:

Maturity level: Draft (for review).

Description

Treatment function code (speciality). This valueset references NHS Data Dictionary: Treatment Function Code.

Note: This valueset defines an inline code system for an NHS code set. This is a temporary provision while an NHS FHIR defined code system becomes available.

Reference: http://www.datadictionary.nhs.uk/data_dictionary/attributes/t/tran/treatment_function_code_de.asp?shownav=0.
