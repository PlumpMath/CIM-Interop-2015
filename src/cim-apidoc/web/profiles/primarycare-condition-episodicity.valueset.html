Value set PrimaryCare-Condition-Episodicity

The official URL for this value set is: 

http://endeavourhealth.org/fhir/ValueSet/primarycare-condition-episodicity

Maturity level: Draft (for review).

Description

The type of relationship that exists between a follow-on condition and the target problem episode.

Definition

This valueset is comprised of the following codes from inline code system http://endeavourhealth.org/fhir/ValueSet/primarycare-condition-episodicity:

CodeDisplayDefinition
reviewReviewThe condition is a review of the target problem episode
flareFlareThe condition marks a flare-up of the target problem episode
endEndThe condition marks the end of the target problem episode
