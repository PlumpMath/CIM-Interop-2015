Overview

The Endeavour Health FHIR Implementation Guide is a collection of profiles, that make conformance rules that, together with textual guidance, provides instruction on how to make use of FHIR in the UK.

This specification is built on FHIR DSTU 2 (1.0.2).

The profiles in this specification are used in the Endeavour Common Interface Mechanism FHIR API.

This work is funded by the Endeavour Health Charitable Trust.
