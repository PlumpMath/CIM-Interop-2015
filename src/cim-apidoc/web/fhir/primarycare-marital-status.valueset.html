Value set PrimaryCare-MaritalStatus

The official URL for this value set is: 

http://endeavourhealth.org/fhir/ValueSet/primarycare-marital-status

Maturity level: Draft (for review).

Description

An indicator to identify the legal marital status of a person. This valueset references NHS Data Dictionary: Person Marital Status Code.

Note: This valueset defines an inline code system for an NHS code set. This is a temporary provision while an NHS FHIR defined code system becomes available.

Reference: http://www.datadictionary.nhs.uk/data_dictionary/attributes/p/person/person_marital_status_code_de.asp?shownav=0.
