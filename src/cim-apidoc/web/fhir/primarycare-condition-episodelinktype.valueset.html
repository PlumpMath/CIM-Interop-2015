Value set PrimaryCare-Condition-EpisodeLinkType

The official URL for this value set is: 

http://endeavourhealth.org/fhir/ValueSet/primarycare-condition-episodelinktype

Maturity level: Draft (for review).

Description

The type of relationship that exists between a condition and the target follow-on observation.

Definition

This valueset is comprised of the following codes from inline code system http://endeavourhealth.org/fhir/ValueSet/primarycare-condition-episodelinktype:

CodeDisplayDefinition
reviewReviewThe target observation is a review of the condition episode
flareFlareThe target observation marks a flare-up of the condition episode
endEndThe target observation marks the end of the condition episode
