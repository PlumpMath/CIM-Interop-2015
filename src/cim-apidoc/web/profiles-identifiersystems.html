Identifier systems

The identifier systems below are for use in the system element of the Identifier data type in Endeavour profiles. Where there is no official identifier system defined by the issuing party, a temporary Endeavour identifier system has been created while an official one becomes available.

Name URI Relevent resource Is temporary? Issuing party reference
NHS number http://fhir.nhs.net/Id/nhs-number PrimaryCare-Patient No HSCIC FHIR API Implementation Guide
CHI number http://endeavourhealth.org/fhir/Identifier/chi-number PrimaryCare-Patient Yes
Health and care number http://endeavourhealth.org/fhir/Identifier/hc-number PrimaryCare-Patient Yes

SDS role profile ID http://fhir.nhs.net/Id/sds-role-profile-id PrimaryCare-Practitioner No HSCIC FHIR API Implementation Guide

e-Referral service UBRN http://endeavourhealth.org/fhir/Identifier/ubrn PrimaryCare-ReferralRequest Yes
